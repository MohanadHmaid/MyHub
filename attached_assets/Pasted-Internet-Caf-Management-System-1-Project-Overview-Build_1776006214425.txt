Internet Café Management System
🧠 1. Project Overview

Build a web-based internet café management system for a single café that supports:

Table visualization (real-time status)
QR code table ordering system
Simple reservation system (no customer accounts)
Admin dashboard for full control
Basic payment tracking (no full payment gateway required)

The system is optimized for simplicity, speed, and real-world usability.

👥 2. User Roles
🟢 Customer (No account required)
View café tables visually
Scan QR code at table
Place orders
Make reservations (guest mode using phone number)
Receive reservation code
🔵 Admin (Login required)
Manage tables
View reservations
Manage orders
Update order status
View payments status
Simple dashboard overview
🧱 3. Core Features
🪑 A. Table System
Each table has:
ID
Name/Number
Status: available | occupied
QR code URL: /table/{tableId}
Visual layout:
Green = available
Red = occupied
📲 B. QR Ordering System

Flow:

Customer scans QR
Opens menu page for that table
Selects items
Places order
Order linked to table ID
🍔 C. Menu & Orders
Product categories (food/drinks)
Add to cart
Submit order
Order statuses:
pending
preparing
completed
📅 D. Reservation System (NO accounts)
Customer inputs:
Name
Phone number
Date/time
System generates:
Reservation code (unique)
Admin can view/manage reservations

Optional:

Mark reservation as confirmed/cancelled
🧾 E. Payment (Simple Version)
Payment status:
unpaid
paid
No payment gateway required (manual confirmation)
🧑‍💻 F. Admin Dashboard (Simple UI)

Sections:

Tables overview (visual grid)
Orders list
Reservations list
Menu management
